Introduction¶

The data analyzed is the data donors file. The file shows donations made with other variables such as income, family size, donation size, and homeownership. The source of this data came directly from the LMS and was imported using pandas into a dataframe.

The data was cleaned by removing columns that wouldn't be used in the analysis. The two row id columns, four zip code columns, and the IC15 column was removed from the dataframe. After removing the columns, the remaining column names were changed to names that were easier to comprehend. After that, the dataframe was checked for NA values; there were no NA's in the data. Finally, the summary statistics were printed to see the averages, mins, maxes, standard deviations, and percentiles of the columns.
